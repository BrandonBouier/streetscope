Geocoding is hard. Geocoders as they exist now are expensive, slow, inefficient, or inaccurate. This is largely because of the data that is available to create these geocoders. But what if there was an easy way to create a geocoder with open data?

Enter Streetscope. Streetscope is an application that creates a geocoder from OpenAddresses data. This tool can be used for any city or county, so long as its data is in OpenAddresses!

If you're interested in setting this up for your city or county, read the instructions on GitHub. If you want to use Streetscope but your data isn't in OpenAddresses, click here.

This project was created by 2014 Code for America Fellows Livien Yin, Erik Schwartz, Lyzi Diamond, and Jeremia Kimelman.
